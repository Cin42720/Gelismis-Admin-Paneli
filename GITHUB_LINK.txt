----------------------------------------
Proje Açıklaması
----------------------------------------

Docker tabanlı, çok gruplu (süpervizör, öğrenci, okul, işletme) bir admin paneli projesidir.
Süpervizör tüm grupların ekranlarını ve CRUD yetkilerini atayıp çıkarabilir.
Her grubun erişebileceği ekranlar ve işlem izinleri (oluşturma, okuma, güncelleme, silme)
dinamik olarak yönetilir.

----------------------------------------
Kullanılan Teknolojiler
----------------------------------------

- Frontend   : React + Vite + TailwindCSS
- Backend    : FastAPI (Python) — Microservice mimarisi
- AI Servis  : FastAPI tabanlı ayrı microservice (RAG / MCP desteği)
- Worker     : RabbitMQ RPC consumer (Python)
- Veritabanı : MySQL 8.4 (JSON veri modeli, Trigger, Function, Procedure)
- Önbellek   : Redis
- Mesajlaşma : RabbitMQ (RPC cevap kuyruğu mantığı)
- Ortam      : Docker Compose (7 container)

----------------------------------------
Servis Adresleri (Docker)
----------------------------------------

- Frontend Panel    : http://localhost:3001
- Backend Swagger   : http://localhost:8000/docs
- AI Servis Swagger : http://localhost:8100/docs
- RabbitMQ Paneli   : http://localhost:15672 (guest / guest)
- MySQL             : localhost:3306

Önemli Özellikler
----------------------------------------

1.  Üç farklı grup girişi (öğrenci, okul, işletme) + Süpervizör yönetimi
2.  Süpervizör ekran ve CRUD yetki atama / çıkarma
3.  Dosya yükleme kısıtlaması (txt, png, jpg, jpeg, pdf, doc, docx, xls, xlsx)
4.  MySQL JSON veri modeli (id, date, value JSON)
5.  Veritabanında 12 Trigger + 1 Function + 2 Procedure
6.  RabbitMQ RPC consumer — işlem sonuçları subscriber'lara bildirilir
7.  Tüm işlemler loglanır (DB trigger + RPC loglama)
8.  Yapay zekâ desteği (dosya analizi, format dönüşümü, RAG/MCP)
9.  Tüm servisler Docker Compose ile ayağa kalkar
10. Örnek veritabanı ve demo kullanıcılar hazır gelir
